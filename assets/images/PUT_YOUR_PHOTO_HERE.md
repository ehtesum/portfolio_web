# Profile Photo

Drop your photo into this folder and name it:

    profile.jpg

Requirements:
- Square crop (e.g. 500 × 500 px)
- JPEG or PNG (if PNG, rename the src in index.html to profile.png)
- Keep file size under 500 KB for fast page loads

The website will automatically display it in the hero and About sections.
If the file is missing, your initials "EUI" are shown as a styled placeholder.
